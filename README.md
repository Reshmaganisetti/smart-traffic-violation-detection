# 🚦 Smart Traffic Violation Detection using AI

## 📌 Overview
This project is an AI-based traffic monitoring system that detects helmet violations, tracks vehicles, and identifies traffic incidents in real-time using computer vision and deep learning.

## 🚀 Features
- Real-time vehicle detection using YOLOv8
- Helmet violation detection
- Multi-object tracking using SORT algorithm
- Traffic density estimation
- Incident detection (accidents, congestion)
- Automated report generation

## 🛠️ Tech Stack
- Python
- YOLOv8
- OpenCV
- Deep Learning
- NumPy, Pandas
- TensorFlow / PyTorch

## 📊 Results
- Achieved ~96% accuracy in helmet detection
- Real-time vehicle tracking and monitoring
